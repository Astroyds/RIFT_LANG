Installation

Install RIFT in minutes

Enterprise-ready installer with checksum verification and PATH integration. Windows installs via WSL for consistent UNIX tooling.

macOS + Linux Native installer
Windows WSL only
Verification Simulated output shown below

Install

Use the official install script for macOS and Linux. On Windows, open a WSL terminal and run the same command.

rift $
Verifying checksum... ok
Installing rift... done
rift version 1.0 ready
curl -fsSL https://rift.astroyds.com/rift/install.sh | sh

Verify

After installation, confirm the CLI is available. Output below is a representative sample.

rift $ rift --version
rift 1.0.0 · checksum verified · build 2026.01

Uninstall

Remove the toolchain cleanly with the official uninstall script.

rift $
Removing rift... done
curl -fsSL https://rift.astroyds.com/rift/uninstall.sh | sh
