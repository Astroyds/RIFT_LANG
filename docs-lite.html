Quickstart

Create a file named hello.rift and run it with the CLI.

let name = "World"
print(`Hello, $@name#!`)
rift $ rift run hello.rift

Syntax overview

Blocks use @ and #. Arrays use ~ and !. Template strings use $@ and # for interpolation.

let values = ~1, 2, 3!
let record = @name: "Ava", active: yes#
print(`User: $@record.name#`)

Control flow

RIFT provides if, while, repeat, and check expressions for deterministic branching.

repeat i in 1..5 @
let output = check yes @
yes when i % 2 == 0 =! "even"
_ =! "odd"
#
print(output)
#

Functions

Functions are declared with conduit. Use give to return values.

conduit fibonacci(n) @
if n ~= 1 @
give n
#
give fibonacci(n - 1) + fibonacci(n - 2)
#

Modules

Use grab to import modules, and share to export symbols.

grab http
grab json
http.get("/api/users", conduit(req) @
let users = ~@id: 1, name: "Ava"#!
give http.json(200, users)
#)

Error handling

RIFT includes try, catch, and finally for structured error handling.

try @
let data = wait fetchUserData(1)
# catch error @
print(`Failed: $@error#`)
#
